## 五、首页布局配置

### 后台操作路径
手机端配置 → 首页DIY → 选择模板或自定义布局

### 推荐布局顺序（从上到下）

1. **轮播图** — 1-3 张（活动封面/品牌介绍）
2. **阶段导航** — 备孕/孕期/产后/全部
3. **推荐活动** — 显示 2-4 个活动卡片
4. **孕育知识** — 显示 2-3 篇文章
5. **推荐商品** — 显示 4-6 个商品
6. **妈妈说** — 横向滚动反馈卡片
7. **邀请有礼** — 邀请码+分享入口
