## 二、隐私保护指引内容

### 2.1 位置信息
- **接口**：wx.getLocation / wx.chooseLocation
- **用途**：活动签到、收货地址选择
- **是否必须**：否，用户可拒绝，不影响其他功能

### 2.2 相册/摄像头
- **接口**：wx.chooseImage / wx.chooseMedia
- **用途**：更换头像、上传反馈图片
- **是否必须**：否，用户可拒绝，不影响其他功能

### 2.3 手机号
- **接口**：button open-type="getPhoneNumber"
- **用途**：登录和身份验证
- **是否必须**：否，用户可选择其他方式登录
