# 孕禧小程序 — 版本说明模板

> 每次提交审核时，在微信后台"版本描述"栏填写以下内容。
> 根据实际版本修改对应部分。

---

## v1.0.0 版本说明

**直接复制到微信后台版本描述：**

```
【版本号】1.0.0
【版本类型】首次发布

【功能概述】
孕禧小程序首次发布，核心功能包括：

1. 首页推荐：根据孕育阶段（备孕/孕期/产后）智能推荐商品、活动、知识文章
2. 活动报名：浏览线下沙龙、孕妈课堂等活动，在线报名
3. 母婴商城：浏览和购买母婴商品
4. 邀请有礼：邀请好友注册获积分奖励（仅一级邀请，积分不可提现）
5. 孕育知识：各阶段专业文章
6. 个人中心：管理孕育阶段、订单、积分等

【登录方式】
微信快捷登录 + 手机号绑定，头像昵称非强制

【隐私说明】
- 收集手机号用于订单联系和活动通知
- 收集孕育阶段用于内容推荐
- 用户可拒绝非必要授权，不影响核心功能
- 详细隐私政策见小程序内"隐私政策"页面

【测试说明】
无需测试账号，微信扫码即可体验全部功能
```
