# 体验版 Smoke Test 路线

> 适用阶段：体验版验收  
> 执行人：测试/产品  
> 输入物：体验版小程序二维码  
> 输出物：验收通过确认（24 项全部 PASS）  
> 最后更新：2026-04-25

---

## 测试链路总览

```
登录 → 阶段设置 → 首页推荐 → 活动报名 → 后台审核/核销 → 商品下单（兜底）→ 邀请绑定 → 妈妈说提交 → 后台审核 → 首页展示
```

---

## ST-01：微信授权登录

| 项目 | 内容 |
|------|------|
| 前置条件 | 小程序已编译上传，体验版二维码可扫码 |
| 操作 | 打开小程序 → 点击"微信登录" → 同意授权 |
| 期望结果 | 跳转到首页，底部 Tab 显示"首页/分类/我的"，"我的"页显示用户头像 |
| 异常分支 | 授权拒绝 → 页面停留在登录页，显示"需要授权才能使用"提示 |
| UAT P0 映射 | 注册登录-微信授权登录 |

## ST-02：协议勾选校验

| 项目 | 内容 |
|------|------|
| 前置条件 | 未登录状态 |
| 操作 | 不勾选协议 → 点击登录按钮 |
| 期望结果 | 弹出提示"请先同意隐私协议"，不发起登录请求 |
| 异常分支 | 无 |
| UAT P0 映射 | 注册登录-协议勾选校验 |

## ST-03：阶段设置

| 项目 | 内容 |
|------|------|
| 前置条件 | 已登录，未设置阶段 |

| 异常分支 | 跳过引导 → 首页展示全阶段内容，"我的"页显示"设置阶段"入口 |
| UAT P0 映射 | 阶段设置-首次引导弹窗 |

## ST-13：邀请关系绑定

| 项目 | 内容 |
|------|------|
| 前置条件 | 用户 A 有邀请码，用户 B 未注册 |
| 操作 | 用户 B 扫描用户 A 的邀请码链接 → 注册登录 |
| 期望结果 | 用户 B 注册成功后，邀请关系自动绑定（InviteReward 表新增记录，trigger_event=register，reward_value=0，status=GRANTED） |
| 异常分支 | 邀请码无效 → 正常注册但不绑定关系 |
| UAT P0 映射 | 邀请-关系绑定 |
