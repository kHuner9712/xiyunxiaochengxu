# 隐私数据映射表

> 本文档说明禧孕小程序收集的母婴相关个人资料字段，包括收集目的、使用场景、是否必填、存储方式、删除/修改方式。
> 依据《个人信息保护法》《信息安全技术 个人信息安全规范》(GB/T 35273) 编写。

## 数据处理架构

```
用户提交 → 后端加密存储 → 数据库
                ↓
         phone_hash（去重）
         phone/name（AES-256-CBC 加密）
                ↓
后台列表 → 脱敏展示（138****1234 / 张*三）
后台详情 → 解密展示（需权限 + 审计日志）
导出 CSV → 解密导出（需权限 + 审计日志）
前端 API → 不返回完整手机号/邮箱
前端缓存 → userStore 排除敏感字段
```

| 字段名 | 数据库列 | 字段类型 | 存储方式 | 去重方式 | 收集目的 | 是否必填 |

## 加密方案

| 项目 | 说明 |
|------|------|
| 加密算法 | AES-256-CBC |
| 密钥来源 | `.env` 的 `MUYING_PRIVACY_KEY`（优先）或 `sxo_config` 的 `muying_privacy_key` |
| 密钥生成 | `php -r "echo bin2hex(openssl_random_pseudo_bytes(32));"` |
| IV | 每次加密随机生成 16 字节，与密文拼接后 Base64 编码存储 |
| 哈希算法 | SHA-256（手机号 + 密钥盐值） |
| 兼容读取 | 先尝试解密，失败则按旧明文读取（`DecryptIfEncrypted`） |

| 表 | 字段 | 加密方式 | 用途 |
|----|------|---------|------|
| `sxo_activity_signup` | `name` | AES-256-CBC | 报名姓名 |
| `sxo_activity_signup` | `phone` | AES-256-CBC | 报名手机号 |
| `sxo_muying_feedback` | `contact` | AES-256-CBC | 联系方式 |

| 表 | 字段 | 哈希方式 | 用途 |

## 审计日志

导出和查看敏感信息时，系统自动记录审计日志到 `sxo_muying_audit_log` 表：

| 字段 | 说明 |
|------|------|
| `admin_id` | 操作管理员 ID |
| `scene` | 操作场景（signup_export/feedback_export/user_export/sensitive_view） |
| `conditions` | 查询条件 JSON |
| `export_count` | 导出/查看数量 |
| `ip` | 操作 IP |
| `add_time` | 操作时间 |

## 前端敏感数据保护

| 措施 | 说明 |
|------|------|
| Personal API 不返回完整手机号 | `mobile` 字段在 API 返回时被移除，仅保留 `mobile_security` |
| Personal API 不返回完整邮箱 | `email` 字段在 API 返回时被移除，仅保留 `email_security` |
| 报名页不预填手机号 | 用户每次报名需手动输入手机号 |
| userStore 排除敏感字段 | `mobile`/`email`/`phone`/`due_date`/`baby_birthday`/`baby_month_age` 不写入本地缓存 |
| 报名成功后仅回填 current_stage | 不缓存 due_date/baby_birthday 到 userStore |

## 旧数据迁移

旧数据（`privacy_version=0`）为明文存储，需要迁移为加密存储。迁移方式：

1. 配置 `MUYING_PRIVACY_KEY` 密钥
2. 运行后端数据迁移脚本（待开发 `scripts/migrate-encrypt-sensitive.php`）
3. 脚本逻辑：读取所有 `privacy_version=0` 的记录 → 加密 name/phone → 生成 phone_hash → 更新 `privacy_version=1`
4. 兼容读取：未迁移的数据，`DecryptIfEncrypted` 会检测到非 Base64 格式，按明文返回

## 用户权利保障

| 权利 | 实现方式 |
|------|---------|
| 知情权 | 报名时展示隐私告知，需勾选同意 |
| 访问权 | 个人中心查看已填写的资料（脱敏展示） |
| 更正权 | 个人中心编辑资料，或联系客服修改 |
| 删除权 | 联系客服删除账户及相关数据（含加密数据） |
| 撤回同意权 | 取消报名即撤回该次活动的信息收集同意 |
