# 一期允许功能范围

> 本文档定义孕禧小程序一期上线允许的功能边界。
> 合规前提：未取得 ICP 经营许可证阶段，只做自营母婴商城 + 官方活动报名 + 会员运营 + 一级邀请裂变。

## 一期允许功能

| 功能模块 | 说明 | 对应插件/模块 |
|---------|------|-------------|
| 自营商城 | 平台自营商品展示、搜索、分类 | 核心商品模块 |
| 商品展示 | 商品详情、图片、规格、库存 | Goods |
| 下单支付 | 购物车、下单、微信支付 | Buy / Cart / Payment |
| 订单售后 | 订单管理、退款退货、物流查询 | Order / Refund |
| 物流跟踪 | 快递信息查询、物流状态展示 | Express |
| 官方活动报名 | 活动列表、详情、报名、取消、签到 | Activity（二开） |
| 会员资料 | 注册、登录、个人信息编辑、母婴画像 | User |
| 积分 | 积分获取与消费记录 | Points（一期基础积分） |
| 优惠券 | 优惠券领取与使用 | Coupon（一期基础优惠券） |
| 一级邀请裂变 | 邀请码生成、一级邀请奖励 | Invite（二开） |
| 官方内容 | 文章、公告、首页装修 | Article / Design / Diy |
| 用户反馈 | 意见反馈提交与查看 | Feedback（二开） |
| 收藏 | 商品收藏、活动收藏 | GoodsFavor |
| 消息通知 | 站内消息 | Message |

## 功能开关机制

高风险插件通过 MyC 配置的功能开关控制，默认关闭：

- 后端拦截：`Plugins::IsPluginBlocked()` 在 API 层强制拦截
- 前端过滤：`phase-one-scope.js` 在导航/路由层隐藏入口
- 配置开关：`feature_xxx_enabled` 在 `sxo_config` 表中管理

开关启用流程：在后台「系统设置 → 配置管理」中设置对应 `feature_xxx_enabled = 1`，前后端同步生效。
