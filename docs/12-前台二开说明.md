# 前台二开说明

## 一、改造范围

本次二开基于 ShopXO v6.8.0 uni-app 端，改造为母婴行业微信商城小程序一期版本。

### 改造原则
1. **基于现有页面和组件改造**，不大拆大重构
2. **新增内容配置化、数据驱动**，方便后续继续改
3. **所有 mock 数据、图片、文案先内置一版演示内容**
4. **母婴主题色独立成套**，不影响原有主题切换功能

## 二、主题色方案

### 2.1 新增主题：muying

| 色阶 | 色值 | 用途 |
|------|------|------|
| 主色 (main) | #F5A0B1 | 按钮、标签、强调文字、导航激活 |
| 次主色 (light) | #FFF0F3 | 浅色背景、卡片底色、标签底色 |
| 搭配色 (pair) | #F5C6A0 | 次要强调、渐变搭配、副标题 |
| 页面背景 | #FFF8F5 | 全局暖白背景 |
| 导航栏背景 | #FFFAF7 | 奶白色导航栏 |

### 2.2 阶段色系

| 阶段 | 主色 | 浅色 | 用途 |
|------|------|------|------|
| 备孕 | #F5A0B1 | #FFF0F3 | 粉色系 |
| 孕期 | #F5C6A0 | #FFF5EC | 杏色系 |
| 产后 | #A0C4F5 | #ECF3FF | 蓝色系 |
| 通用 | #A0D8B0 | #ECFFF3 | 绿色系 |

### 2.3 使用方式

在 CSS 中使用 `.theme-muying` 前缀的类名：
```html
<view class="bg-main cr-main">主色背景+文字</view>
<view class="bg-main-light cr-main">浅色背景+主色文字</view>
```

在母婴专属样式中使用 `.muying-*` 前缀：
```html
<view class="muying-card">母婴风格卡片</view>
<view class="muying-stage-tag muying-stage-tag--pregnancy">孕期</view>
```

## 七、编译运行

### 7.1 HBuilderX 运行
1. 打开 HBuilderX
2. 文件 → 打开目录 → 选择 `shopxo-uniapp`
3. 运行 → 运行到小程序模拟器 → 微信开发者工具
4. 或 运行 → 运行到浏览器 → Chrome（H5模式）

### 7.2 注意事项
- 首次编译可能较慢，因为新增了多个页面和组件
- 微信小程序开发者工具中需关闭"不校验合法域名"选项
- 活动Tab图标为占位图，功能正常但视觉需优化
