# 新增字段与数据结构

## 一、数据库新增表

### 1.1 sxo_activity（活动表）

| 列名 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| id | int unsigned AUTO_INCREMENT | - | 自增ID |
| title | char(200) | '' | 活动标题 |
| cover | char(255) | '' | 封面图 |
| images | text | NULL | 相册图片JSON |
| category | char(30) | '' | 活动类型(classroom/salon/trial/lecture) |
| stage | char(30) | '' | 适用阶段(prepare/pregnancy/postpartum/all) |
| description | text | NULL | 活动简介 |
| content | longtext | NULL | 活动详情HTML |
| address | char(255) | '' | 活动地址 |
| start_time | int unsigned | 0 | 开始时间戳 |
| end_time | int unsigned | 0 | 结束时间戳 |
| signup_start_time | int unsigned | 0 | 报名开始时间戳 |
| signup_end_time | int unsigned | 0 | 报名截止时间戳 |
| max_count | int unsigned | 0 | 最大报名人数(0不限) |
| signup_count | int unsigned | 0 | 已报名人数 |
| is_free | tinyint unsigned | 1 | 是否免费(0否/1是) |
| price | decimal(10,2) | 0.00 | 活动价格 |
| contact_name | char(60) | '' | 联系人 |
| contact_phone | char(30) | '' | 联系电话 |
| access_count | int unsigned | 0 | 访问量 |
| sort_level | int unsigned | 0 | 排序值 |
| is_enable | tinyint unsigned | 1 | 是否启用(0否/1是) |
| is_delete_time | int unsigned | 0 | 删除时间(0未删除) |
| add_time | int unsigned | 0 | 添加时间戳 |
| upd_time | int unsigned | 0 | 更新时间戳 |

**索引**：PRIMARY(id), KEY(category), KEY(stage), KEY(is_enable), KEY(is_delete_time)

**category 枚举值**：
- `classroom` - 孕妈课堂
- `salon` - 线下沙龙
- `trial` - 试用活动
- `lecture` - 讲座报名

**stage 枚举值**：
- `prepare` - 备孕
- `pregnancy` - 孕期
- `postpartum` - 产后
- `all` - 通用

### 1.2 sxo_activity_signup（活动报名表）

| 列名 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| id | int unsigned AUTO_INCREMENT | - | 自增ID |
| activity_id | int unsigned | 0 | 活动ID |
| user_id | int unsigned | 0 | 用户ID |
| name | char(60) | '' | 报名姓名 |
| phone | char(30) | '' | 联系电话 |
| stage | char(30) | '' | 当前阶段 |
| remark | char(255) | '' | 备注 |
| status | tinyint unsigned | 0 | 状态(0待确认/1已确认/2已取消) |
| is_delete_time | int unsigned | 0 | 删除时间 |
| add_time | int unsigned | 0 | 添加时间戳 |
| upd_time | int unsigned | 0 | 更新时间戳 |

**索引**：PRIMARY(id), KEY(activity_id), KEY(user_id), KEY(status)

### 1.3 sxo_invite_reward（邀请奖励表）

| 列名 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| id | int unsigned AUTO_INCREMENT | - | 自增ID |
| inviter_id | int unsigned | 0 | 邀请人用户ID |
| invitee_id | int unsigned | 0 | 被邀请人用户ID |
| reward_type | char(30) | 'integral' | 奖励类型(integral/coupon) |
| reward_value | int unsigned | 0 | 奖励值(积分数/优惠券ID) |
| trigger_event | char(30) | 'register' | 触发事件(register/first_order) |
| status | tinyint unsigned | 0 | 状态(0待发放/1已发放/2已取消) |
| add_time | int unsigned | 0 | 添加时间戳 |
| upd_time | int unsigned | 0 | 更新时间戳 |

**索引**：PRIMARY(id), KEY(inviter_id), KEY(invitee_id), KEY(trigger_event), KEY(status)

**trigger_event 枚举值**：
- `register` - 新用户注册
- `first_order` - 被邀请人首单完成

**reward_type 枚举值**：
- `integral` - 积分奖励
- `coupon` - 优惠券奖励

## 三、后端新增API

### 3.1 活动API（Activity控制器）

| 方法 | 路径 | 需登录 | 说明 |
|------|------|--------|------|
| Index | /api.php?s=activity/index | 否 | 活动列表（支持category/stage/is_free筛选，分页） |
| Detail | /api.php?s=activity/detail | 否 | 活动详情（含访问量自增） |
| Signup | /api.php?s=activity/signup | 是 | 活动报名（校验时间段/人数/重复） |
| SignupCancel | /api.php?s=activity/signupcancel | 是 | 取消报名 |
| MySignup | /api.php?s=activity/mysignup | 是 | 我的报名记录 |

### 3.2 邀请API（Invite控制器）

| 方法 | 路径 | 需登录 | 说明 |
|------|------|--------|------|
| Index | /api.php?s=invite/index | 是 | 邀请统计（人数/奖励/邀请码） |
| RewardList | /api.php?s=invite/rewardlist | 是 | 奖励记录（分页） |
| Poster | /api.php?s=invite/poster | 是 | 邀请海报数据 |

### 3.3 个人资料API修改

| 方法 | 路径 | 修改内容 |
|------|------|----------|
| Save | /api.php?s=personal/save | 支持current_stage/due_date/baby_birthday字段 |

## 四、前端数据结构

### 4.1 首页母婴模块数据

```javascript
// 活动列表
muying_activity_list: [{
  id: Number,
  title: String,        // 活动标题
  cover: String,        // 封面图URL
  time_text: String,    // 时间文本
  stage: String,        // prepare/pregnancy/postpartum
  stage_text: String,   // 备孕/孕期/产后
  is_free: Boolean,
  price: String
}]

// 阶段筛选
muying_stage_tabs: [{
  name: String,   // 全部/备孕/孕期/产后
  value: String   // ''/prepare/pregnancy/postpartum
}]

// 文章资讯
muying_article_list: [{
  id: Number,
  title: String,
  desc: String,
  tags: Array<String>
}]

// 用户反馈
muying_feedback_list: [{
  avatar_emoji: String,  // emoji头像
  name: String,
  content: String,
  stage_text: String
}]
```
