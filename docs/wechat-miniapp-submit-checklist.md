# 禧孕 V1.0 微信小程序提审检查清单

## 1. 提审前必检项

### 1.1 资质与备案

| 项目 | 要求 | 当前状态 |
|------|------|---------|
| ICP 备案 | 域名已通过管局审核 | ☐ |
| SSL 证书 | HTTPS 正常访问 | ☐ |
| 小程序主体 | 已注册企业主体 | ☐ |
| 正式 AppID | 已申请并配置 | ☐ |

### 1.2 功能开关

| 项目 | 要求 | 检查方式 |
|------|------|---------|
| 高风险功能开关 | 全部关闭 | 后台 → 合规中心 → 功能开关 |
| 资质门禁 | 正确配置 | 后台 → 合规中心 → 资质状态 |
| 一期核心功能 | 正常可用 | 活动报名/邀请/内容/反馈 |

### 1.3 隐私与权限

| 项目 | 要求 | 说明 |
|------|------|------|
| 用户隐私协议 | 已配置 | 后台 → 协议管理 |
| 隐私弹窗 | 正常弹出 | 首次使用敏感API时弹出 |
| requiredPrivateInfos | 一期为空 | 一期不强制定位，后续活动签到需定位时独立开启 |
| 定位权限 | 一期不申请 | 后续活动签到/地址选择时按需开启，需同步更新 requiredPrivateInfos 和 permission |

### 1.4 内容合规

| 项目 | 要求 | 说明 |
|------|------|------|
| 无 UGC 内容 | 不允许用户发帖/评论 | ask/blog 已屏蔽 |
| 无医疗内容 | 不提供问诊/咨询 | hospital 已屏蔽 |
| 无直播内容 | 不提供直播功能 | weixinliveplayer 已屏蔽 |
| 商品描述合规 | 无虚假宣传 | 人工检查 |
| 无分销/返佣 | 不提供多级分销 | distribution 已屏蔽 |

## 2. 微信后台配置

### 2.1 服务器域名

登录 [微信公众平台](https://mp.weixin.qq.com) → 开发管理 → 开发设置：

| 域名类型 | 填写 | 说明 |
|---------|------|------|
| request 合法域名 | `https://你的备案域名` | API 请求 |
| uploadFile 合法域名 | `https://你的备案域名` | 文件上传 |
| downloadFile 合法域名 | `https://你的备案域名` | 文件下载 |

### 2.2 隐私设置

开发管理 → 开发设置 → 隐私与安全：
- 用户隐私保护指引：填写并提交
- 隐私弹窗：开启

### 2.3 版本管理

版本管理 → 开发版本：
- 确认版本号正确
- 确认体验版可正常使用
- 点击"提交审核"

## 3. 提审信息填写

### 3.1 服务类目

选择：`商业服务 → 电商` 或 `生活服务 → 母婴`

### 3.2 测试账号

如需提供测试账号：
- 账号：`admin`
- 密码：（临时密码，审核通过后修改）

### 3.3 版本描述

```
禧孕母婴商城 V1.0
- 自营母婴商品浏览与购买
- 官方活动报名与签到
- 孕育阶段个性化推荐
- 一级邀请留资
- 用户反馈与妈妈说
```

## 4. 审核常见被拒原因与规避

| 被拒原因 | 规避方式 |
|---------|---------|
| 虚拟支付/充值 | 钱包/余额/充值/提现已屏蔽 |
| 多商户/平台型 | shop/realstore 已屏蔽 |
| 分销/返佣 | distribution 已屏蔽 |
| UGC 内容审核 | ask/blog 已屏蔽 |
| 医疗问诊 | hospital 已屏蔽 |
| 直播 | weixinliveplayer 已屏蔽 |
| 隐私协议缺失 | 确保隐私协议已配置 |
| 定位权限滥用 | 一期不申请定位权限 | 后续按需开启 |
| 测试号/AppID 问题 | 使用正式 AppID |

## 5. 提审后监控

1. 关注微信公众平台审核通知
2. 如被拒，根据原因修改后重新提交
3. 审核通过后，确认线上版本功能正常
4. 修改默认管理员密码
5. 确认合规中心所有开关状态正确
