# 禧孕母婴用品私域商城小程序 - 小程序页面设计

## 1. 文档概述

本文档为"禧孕母婴用品私域商城小程序"的小程序页面设计文档，由禧孕文化传媒有限公司运营，属于甲方自营商城（非多商户平台）。

技术栈：uni-app + Vue3 + TypeScript

本文档详细定义了小程序所有页面的路径、用途、核心组件、用户操作、调用接口、页面状态、异常情况、空状态及加载状态，为前端开发提供明确的实现依据。

### 1.1 文档范围

- 涵盖小程序全部 27 个页面的详细设计
- 定义页面间跳转关系
- 定义全局复用组件
- 定义页面配置规范

### 1.2 读者对象

- 前端开发工程师
- 后端开发工程师
- 产品经理
- UI 设计师
- 测试工程师

### 1.3 术语说明

| 术语 | 说明 |
|------|------|
| SKU | Stock Keeping Unit，库存量单位，对应商品的具体规格组合 |
| SPU | Standard Product Unit，标准产品单位，对应一个商品 |
| 月龄 | 宝宝的月龄，用于按月龄推荐商品 |
| 积分 | 用户通过购物、签到等行为获得的虚拟积分，可用于抵扣 |
| 优惠券 | 由平台发放的优惠凭证，满足条件后可抵扣订单金额 |
| 售后 | 订单完成后的退换货服务 |
| 骨架屏 | 页面加载时展示的内容占位组件，提升用户感知速度 |

### 2.3 登录拦截说明

- 标记为"需要登录"的页面，在用户未登录时自动跳转至登录流程（微信授权登录）
- 标记为"不需要登录"的页面，未登录用户可正常浏览，但涉及用户身份的操作（如加购、收藏）需引导登录
- 首页、分类页、商品详情页、搜索页、活动页、内容页支持未登录浏览，部分功能（月龄推荐、收藏、加购）需登录后使用

## 3. 逐页详细设计

### 3.1 首页 /pages/home/index

| 属性 | 说明 |
|------|------|
| 页面路径 | /pages/home/index |
| 页面用途 | 商城首页，展示Banner、推荐位、活动入口、热门商品、新品推荐、按月龄推荐 |
| 所属Tab | 首页Tab |

**核心组件：**

| 组件名 | 说明 |
|--------|------|
| SearchBar | 顶部搜索栏，点击跳转搜索页，不支持直接输入 |
| BannerSwiper | 轮播Banner，支持自动播放、指示器、点击跳转 |
| QuickNav | 功能快捷入口（8个图标入口：新品、活动、优惠券、积分、会员、育儿知识、宝宝档案、客服） |
| ActivityEntry | 活动入口卡片，展示进行中的重点活动 |
| AgeRecommend | 按宝宝月龄推荐商品，需登录且有宝宝档案时展示 |
| HotProducts | 热门商品横向滚动列表 |
| NewProducts | 新品推荐网格列表 |
| GuessLike | 猜你喜欢瀑布流列表，支持下拉加载更多 |

**用户操作：**

| 操作 | 触发方式 | 响应 |
|------|----------|------|
| 点击搜索栏 | tap | 跳转搜索页 /pages/search/index |
| 点击Banner | tap | 根据Banner配置跳转（商品详情/活动页/内容页/外部链接） |
| 点击快捷入口 | tap | 跳转对应功能页 |
| 点击活动入口 | tap | 跳转活动详情 /pages/activity/detail |
| 点击月龄推荐商品 | tap | 跳转商品详情 /pages/product/detail |
| 点击热门/新品商品 | tap | 跳转商品详情 /pages/product/detail |
| 下拉刷新 | pullDownRefresh | 重新加载首页数据 |
| 上拉加载更多 | reachBottom | 加载更多猜你喜欢商品 |
| 分享 | onShareAppMessage | 分享小程序卡片到微信聊天 |

**调用接口：**

| 接口 | 方法 | 说明 | 请求参数 | 返回数据 |
|------|------|------|----------|----------|
| /api/weapp/home/data | GET | 获取首页数据（Banner、快捷入口、活动入口、热门、新品） | 无 | banners, quickNav, activities, hotProducts, newProducts |
| /api/weapp/product/recommend | GET | 获取推荐商品（猜你喜欢/月龄推荐） | type: guess/age, page, pageSize | list, total, hasMore |

**页面状态：**

| 状态 | 说明 | 展示内容 |
|------|------|----------|
| 正常状态 | 已登录且有宝宝档案 | 展示全部模块，含月龄推荐 |
| 未登录状态 | 用户未登录 | 隐藏月龄推荐模块，猜你喜欢展示通用推荐 |
| 已登录无宝宝 | 已登录但未添加宝宝档案 | 月龄推荐模块显示"添加宝宝档案"引导 |

**异常情况：**

| 异常 | 处理方式 |
|------|----------|
| 网络错误 | 展示网络错误提示，提供"重新加载"按钮 |
| 接口超时 | 展示加载超时提示，提供"重新加载"按钮 |
| 数据为空 | 各模块独立处理，某个模块数据为空时不影响其他模块展示 |

**空状态：**

| 场景 | 展示内容 |
|------|----------|
| 热门商品为空 | 隐藏热门商品模块 |
| 新品推荐为空 | 隐藏新品推荐模块 |
| 猜你喜欢为空 | 展示"暂无推荐商品"空状态图 |
| 活动入口为空 | 隐藏活动入口模块 |

**加载状态：**

| 场景 | 展示方式 |
|------|----------|
| 首次进入 | 骨架屏加载，按模块依次展示骨架占位 |
| 下拉刷新 | 顶部刷新指示器 |
| 上拉加载更多 | 底部加载指示器（加载中/没有更多了） |

### 3.4 商品详情 /pages/product/detail

| 属性 | 说明 |
|------|------|
| 页面路径 | /pages/product/detail |
| 页面用途 | 展示商品完整信息，支持SKU选择、加购、收藏、分享 |
| 所属Tab | 无 |

**核心组件：**

| 组件名 | 说明 |
|--------|------|
| ImageSwiper | 商品图片轮播，支持指示器和手动滑动 |
| PriceInfo | 价格信息区域，展示售价、原价、会员价、促销标签 |
| SkuSelector | SKU选择弹窗，选择规格、数量 |
| ProductInfo | 商品基本信息（名称、描述、销量、运费） |
| RichTextDetail | 商品图文详情，富文本渲染 |
| AttrTable | 商品参数表格 |
| ServicePromise | 服务承诺标签（正品保证、七天退换、极速发货等） |
| ReviewList | 商品评价列表摘要，展示最新3条评价 |
| BottomActionBar | 底部操作栏（客服、收藏、加入购物车、立即购买） |
| SharePopup | 分享弹窗（分享好友、生成海报） |

**用户操作：**

| 操作 | 触发方式 | 响应 |
|------|----------|------|
| 滑动图片 | swipe | 切换商品图片 |
| 点击规格选择 | tap | 弹出SKU选择弹窗 |
| 选择SKU规格 | tap | 更新价格、库存、图片 |
| 修改数量 | tap +/- | 更新数量，校验库存 |
| 加入购物车 | tap | 校验登录 → 校验SKU → 调用加购接口 → 成功提示 |
| 立即购买 | tap | 校验登录 → 校验SKU → 跳转下单确认页 |
| 收藏/取消收藏 | tap | 调用收藏接口，切换收藏状态 |
| 分享 | tap | 弹出分享弹窗 |
| 查看全部评价 | tap | 跳转评价列表（如有独立页面） |
| 客服 | tap | 跳转客服会话 |

**调用接口：**

| 接口 | 方法 | 说明 | 请求参数 | 返回数据 |
|------|------|------|----------|----------|
| /api/weapp/product/detail | GET | 获取商品详情 | id | product{...}, skus[{...}], reviews[{...}] |
| /api/weapp/cart/add | POST | 加入购物车 | productId, skuId, quantity | cartItemId |
| /api/weapp/favorite/toggle | POST | 切换收藏状态 | productId | isFavorited |

**页面状态：**

| 状态 | 说明 | 展示内容 |
|------|------|----------|
| 正常状态 | 商品在售且有库存 | 完整商品信息，底部操作栏正常 |
| 已下架 | 商品已下架 | 展示商品信息，底部显示"商品已下架"，禁用加购和购买 |
| 库存不足 | SKU库存为0 | SKU选择时标注"库存不足"，加购按钮置灰 |

**异常情况：**

| 异常 | 处理方式 |
|------|----------|
| 商品不存在 | 展示"商品不存在"提示，提供"返回首页"按钮 |
| 商品已删除 | 展示"商品已下架"提示，提供"返回首页"按钮 |
| 加载失败 | 展示错误提示，提供"重新加载"按钮 |

**空状态：**

| 场景 | 展示内容 |
|------|----------|
| 不适用 | 商品详情页不存在空状态，异常情况走错误处理 |

**加载状态：**

| 场景 | 展示方式 |
|------|----------|
| 商品信息加载中 | 骨架屏加载（图片区域 + 信息区域骨架） |

### 5.2 组件设计原则

| 原则 | 说明 |
|------|------|
| 单一职责 | 每个组件只负责一个功能点 |
| 可配置性 | 通过Props传入配置，支持不同场景的展示需求 |
| 可复用性 | 设计为全局组件，避免页面内重复实现 |
| 样式一致性 | 统一设计规范，颜色、字号、间距保持一致 |
| 类型安全 | 使用TypeScript定义Props和Emits类型 |
| 响应式 | 适配不同屏幕尺寸 |
