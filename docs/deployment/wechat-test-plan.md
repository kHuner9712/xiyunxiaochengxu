# 微信小程序测试号联调清单

> 面向：孕禧小程序一期，使用微信测试号联调
> 前提：后端已部署，数据库已迁移，Nginx 已配置

---

## 1. 启动与基础配置

| 步骤 | 操作 | 预期结果 | 失败排查 |
|------|------|---------|---------|
| 1.1 | 微信开发者工具导入项目，填入测试号 AppID | 项目正常加载 | 检查 AppID 是否正确，项目路径是否包含 manifest.json |
| 1.2 | 勾选「不校验合法域名、web-view、TLS版本」 | 可请求 HTTP 接口 | 设置→项目设置→本地设置 |
| 1.3 | 配置 `.env.development` 中 `UNI_APP_REQUEST_URL` 为后端地址 | 首页能加载 | 检查后端是否启动，URL 末尾是否有 `/` |
| 1.4 | 编译运行到模拟器 | 首页正常显示 | 查看控制台报错 |
