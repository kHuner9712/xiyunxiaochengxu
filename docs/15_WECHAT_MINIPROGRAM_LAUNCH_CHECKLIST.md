# 微信小程序上线配置清单

## 1. 小程序基本信息

- 小程序 AppID / AppSecret（在微信公众平台 → 开发管理 → 开发设置 获取）
- 小程序名称、简介、类目
- 小程序头像（1024x1024）

## 2. 小程序类目

- 必须选择与母婴用品电商相关的类目
- 推荐：商家自营 → 母婴用品 / 电商 → 综合电商
- 需要提供相关经营资质

## 3. 微信支付配置

- 商户号 (mchId)
- API v3 密钥
- 商户私钥文件 (apiclient_key.pem)
- 平台证书
- 证书序列号
- 支付回调域名 (WECHAT_NOTIFY_URL)
- 退款回调域名 (WECHAT_REFUND_NOTIFY_URL)
- 回调URL必须为HTTPS

## 4. 域名配置（微信公众平台 → 开发管理 → 开发设置）

- request 合法域名：API 服务器域名（如 https://api.example.com）
- uploadFile 合法域名：文件上传域名（如 https://api.example.com）
- downloadFile 合法域名：文件下载域名
- 所有域名必须为 HTTPS

## 5. 隐私协议与用户协议

- 隐私保护指引（必须在微信公众平台填写）
- 用户服务协议
- 收集用户信息声明（手机号、地址、宝宝信息等）
- 微信小程序隐私协议更新（2023年9月起生效的隐私新规）

## 6. 经营资质

- 营业执照
- ICP 备案号
- 小程序备案（2023年9月起强制要求）
- 母婴用品经营许可/品牌授权
- 食品经营许可证（如涉及婴幼儿食品）
- 特殊化妆品备案（如涉及婴幼儿护肤品）

## 7. 客服与售后

- 客服入口（小程序内必须提供客服入口）
- 售后规则说明
- 退款规则说明
- 发货规则说明
- 7天无理由退货政策

## 8. 测试账号与审核说明

- 提供测试账号（手机号+验证码或账号密码）
- 审核说明文档（说明小程序功能和使用流程）
- 如有支付功能，提供测试支付金额和退款流程
- 审核期间保持测试账号可用

## 9. 安全配置

- 服务器域名白名单
- 业务域名配置
- Webview 域名（如使用）
- 内容安全检测（文本/图片）

## 10. 上线前检查

- [ ] 小程序信息完整
- [ ] 类目和资质齐全
- [ ] 域名配置正确
- [ ] 隐私协议已填写
- [ ] 客服入口可用
- [ ] 支付功能正常
- [ ] 测试账号可用
- [ ] 审核说明完整
- [ ] ICP/小程序备案完成
- [ ] 真机测试通过
