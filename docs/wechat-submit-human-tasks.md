## 二、逐项说明

### 2.1 正式小程序 AppID 申请

| 项目 | 说明 |
|------|------|
| 办理入口 | https://mp.weixin.qq.com → 注册小程序账号 |
| 所需材料 | 营业执照、法人身份证、对公账户信息 |
| 注意事项 | 注册主体必须与后续 ICP 备案主体一致 |
| 产出 | 正式 AppID（替换测试号 `wxda7779770f53e901`） |
| 配置位置 | `shopxo-uniapp/.env.production` → `UNI_APP_WX_APPID` |

### 2.2 小程序主体认证

| 项目 | 说明 |
|------|------|
| 办理入口 | 微信公众平台 → 设置 → 微信认证 |
| 费用 | 300 元/年 |
| 所需材料 | 营业执照、对公账户、法人信息、认证公函 |
| 审核周期 | 1-5 个工作日 |
| 注意事项 | 认证通过后才能使用支付、获取用户手机号等能力 |

### 2.3 小程序备案

| 项目 | 说明 |
|------|------|
| 办理入口 | 微信公众平台 → 设置 → 小程序备案 |
| 前置条件 | 主体认证已通过 |
| 所需材料 | 营业执照、法人身份证、域名证书、服务器接入证明 |
| 审核周期 | 5-20 个工作日 |
| ⚠️ 强制要求 | 2023年9月起，未备案小程序无法上架 |

### 2.4 域名 ICP 备案

| 项目 | 说明 |
|------|------|
| 办理入口 | 云服务商备案系统（阿里云/腾讯云/华为云等） |
| 所需材料 | 营业执照、法人身份证、域名证书、服务器购买凭证 |
| 审核周期 | 5-20 个工作日 |
| 注意事项 | 备案主体必须与小程序主体一致 |
| 产出 | ICP 备案号（如 京ICP备XXXXXXXX号） |

### 2.5 HTTPS 证书

| 项目 | 说明 |
|------|------|
| 办理方式 A | 宝塔 → 网站 → SSL → Let's Encrypt（免费，自动续期） |
| 办理方式 B | 购买商业证书（OV/EV，审核更严格但更可信） |
| 前置条件 | 域名已解析到服务器、ICP 备案已通过 |
| 验证方式 | 浏览器访问 `https://你的域名/` 无证书警告 |
| ⚠️ 强制要求 | 微信小程序所有请求必须走 HTTPS |

### 2.6 微信服务器域名配置

> ⚠️ 域名必须 ICP 备案通过 + HTTPS 正常后才能配置

| 域名类型 | 填写内容 | 说明 |
|----------|----------|------|
| request 合法域名 | `https://你的API域名` | API 请求地址 |
| uploadFile 合法域名 | `https://你的API域名` | 文件上传地址（与 API 相同） |
| downloadFile 合法域名 | `https://你的API域名` | 文件下载地址（与 API 相同） |
| socket 合法域名 | **不填** | 一期无 WebSocket 功能 |

配置入口：微信公众平台 → 开发管理 → 开发设置 → 服务器域名

### 2.7 微信隐私保护指引

| 项目 | 说明 |
|------|------|
| 配置入口 | 微信公众平台 → 设置 → 基本设置 → 用户隐私保护指引 |
| 必填项 | 根据小程序实际收集的信息勾选 |

一期需声明的信息收集项：

| 信息类型 | 用途 | 是否必填 |
|----------|------|----------|
| 手机号 | 登录、活动报名联系 | 是 |
| 昵称/头像 | 用户身份展示 | 是 |
| 位置信息 | 活动签到、收货地址 | 是 |
| 相册/摄像头 | 更换头像、提交反馈图片 | 是 |
| 设备信息 | 保障服务正常运行 | 是 |

⚠️ 一期**不收集**以下信息（不要勾选）：
- 微信步数
- 通讯录
- 蓝牙
- 日历
- 健康数据

### 2.10 服务类目选择

> ⚠️ 服务类目直接影响审核通过率，选错类目会被驳回

**推荐类目**（根据一期功能）：

| 优先级 | 类目 | 理由 |
|--------|------|------|
| 🟢 首选 | **生活服务 → 母婴服务** | 核心功能为母婴活动报名/内容展示 |
| 🟢 首选 | **商业服务 → 品牌/产品展示** | 品牌展示 + 商品浏览 |
| 🟡 备选 | **生活服务 → 综合生活服务** | 如母婴类目审核有疑问可换此 |
| 🟡 备选 | **工具 → 信息查询** | 活动查询/报名 |

**禁止选择**的类目（一期无资质/无功能）：

| ❌ 禁选类目 | 原因 |
|-------------|------|
| 医疗保健 → 互联网医院 | 无医疗机构执业许可证 |
| 医疗保健 → 医疗问诊 | 无医疗资质 |
| 医疗保健 → 药品/医疗器械 | 无相关资质 |
| 社交 → 直播 | 无网络文化经营许可证 |
| 电商 → 电商平台 | 无 ICP 经营许可证 + EDI 许可证，且一期不开放多商户 |
| 金融理财 → 任何子类 | 无金融资质 |

## 三、审核备注文案（可直接复制）

```
【小程序简介】
本小程序"孕禧"为母婴生活服务平台，主要提供母婴活动报名、品牌商品浏览、
孕育知识内容等功能。

【功能说明】
1. 首页：展示推荐商品、活动、文章等内容
2. 商品浏览：支持商品分类、搜索、详情查看、加入购物车
3. 活动报名：支持查看活动详情、在线报名（需填写姓名/手机/孕育阶段），
   报名时需勾选隐私告知（必选），画像同步授权为可选
4. 内容资讯：母婴知识文章浏览
5. 意见反馈：用户可提交反馈意见
6. 个人中心：订单查看、地址管理、收藏等基础功能

【重要声明】
1. 本小程序不提供医疗诊断、问诊、用药建议等医疗服务
2. 本小程序不开放多商户入驻、分销推广、钱包充值、积分商城、
   优惠券、直播、签到等功能
3. 当前线上支付功能暂未开放，用户可浏览商品和创建待支付订单，
   但无法完成在线支付
4. 活动报名中的"孕育阶段"信息仅用于活动推荐，不用于医疗诊断
5. 活动详情页底部已标注免责声明："本活动不构成医疗建议，
   如有健康疑问请咨询专业医疗机构"

【测试指引】
1. 首页可浏览推荐内容
2. 点击"分类"可查看商品列表
3. 点击"活动"可查看活动列表，选择活动进入详情页可查看报名信息
4. 点击"我的"可查看个人中心
5. 支付相关页面（收银台/支付提示）因支付未开放会提示"线上支付暂未开放"

【隐私与合规】
1. 隐私政策：小程序内"个人中心→隐私政策"可查看完整隐私政策
2. 用户协议：小程序内"个人中心→用户协议"可查看完整用户协议
3. 隐私保护指引：已在微信公众平台配置
4. 活动报名隐私告知：报名时必须勾选同意隐私告知，画像同步为可选项
5. 敏感信息加密：用户手机号、姓名等敏感信息采用 AES-256-CBC 加密存储
```

## 四、办理顺序建议

```
1. 营业执照准备（如已有则跳过）
   ↓
2. 域名 ICP 备案（5-20个工作日，最耗时，优先启动）
   ↓
3. 微信小程序注册 + 主体认证（1-5个工作日）
   ├── 同步进行：小程序备案（5-20个工作日）
   └── 同步进行：用户协议/隐私政策法务审核
   ↓
4. HTTPS 证书申请（ICP 备案通过后，1天内）
   ↓
5. 微信服务器域名配置（HTTPS 正常后，即时生效）
   ↓
6. 微信隐私保护指引配置（即时）
   ↓
7. 服务类目选择（即时，但需谨慎）
   ↓
8. 客服绑定 + 测试账号准备（即时）
   ↓
9. HBuilderX 正式构建 + 上传代码
   ↓
10. 填写审核备注 → 提交审核
```

## 五、关键风险提示

| # | 风险 | 影响 | 应对 |
|---|------|------|------|
| 1 | ICP 备案主体与小程序主体不一致 | 备案被驳回，无法配置服务器域名 | 注册时确认主体一致 |
| 2 | 选了医疗/电商类目 | 审核驳回，要求补充资质 | 严格按推荐类目选择 |
| 3 | 隐私保护指引未配置 | 审核驳回 | 提审前确认已配置 |
| 4 | 服务器域名未配 HTTPS | 无法配置合法域名 | ICP 备案后立即申请证书 |
| 5 | 审核期间客服无响应 | 审核超时或驳回 | 绑定在线客服微信号 |
| 6 | 用户协议/隐私政策未法务审核 | 合规风险 | 提审前完成法务审核 |
| 7 | 正式 AppID 未申请就提审 | 使用测试号提审会被驳回 | 确认 AppID 为正式号 |
