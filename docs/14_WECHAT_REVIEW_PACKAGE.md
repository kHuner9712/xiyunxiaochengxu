## 3. 核心功能说明

### 3.1 母婴用品自营商城
- 本小程序为自营商城，非平台模式
- 所有商品由禧孕母婴统一采购和发货
- 支持快递配送和到店自提两种配送方式

### 3.2 商品浏览
- 首页展示 Banner、推荐商品
- 分类浏览商品
- 搜索商品
- 商品详情页展示 SKU、价格、库存

### 3.3 下单支付
- 支持微信支付
- 支持优惠券和积分抵扣
- 支持快递配送和到店自提

### 3.4 到店自提
- 用户下单时选择"到店自提"
- 选择自提点
- 支付成功后生成6位数字自提码
- 到店出示自提码核销取货

### 3.5 客服与帮助
- 小程序内提供客服入口（电话/微信/FAQ）
- 客服信息由后台配置，非固定联系方式
- 用户可在商品详情、订单详情、个人中心等页面联系客服

### 3.6 文章/视频内容
- 活动板块提供文章和视频内容
- 内容为母婴育儿相关知识分享
- 内容可关联推荐商品

### 3.7 售后退款
- 用户可对符合条件的订单申请售后
- 后台审核后发起退款
- 退款原路返回微信支付
