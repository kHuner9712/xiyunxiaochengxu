# 项目现状总览

## 一、项目定位

**云禧母婴微信商城小程序一期 MVP**

面向母婴行业的一期最小可行产品，基于 ShopXO 开源电商系统二次开发，核心实现母婴阶段化运营（备孕/孕期/产后）、线下活动报名与核销、一级邀请裂变三大业务模块。

### 一期 MVP 范围

- 母婴阶段化首页与商品分类
- 线下活动发布、报名、签到核销
- 一级邀请裂变（注册奖励 + 首单奖励）
- 后台运营管理（活动/报名/邀请/数据报表）
- 微信小程序端 + H5 端

### 一期 MVP 不包含

- 多级分销
- 直播
- 多商户
- 积分商城兑换
- 优惠券发放（仅积分奖励）

## 二、技术栈

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 后端框架 | ShopXO | v6.8.0 | 基于 ThinkPHP 8 的开源电商系统 |
| 后端语言 | PHP | 8.1 | PHP-FPM 运行模式 |
| 后端框架内核 | ThinkPHP | 8.x | ShopXO 底层框架 |

| 数据库 | MySQL | 5.7+ | utf8mb4 字符集（推荐 8.0） |
| Web 服务器 | Nginx | 1.25-alpine | 反向代理 + rewrite |
| 容器化 | Docker + Docker Compose | 3.8 | 本地开发与部署 |
| 包管理 | Composer / npm | - | 后端/前端依赖管理 |
| 编译工具 | HBuilderX | 3.6+ | uni-app 编译运行 |
