## 二、启动环境

### 2.1 启动 Docker 容器

```powershell
cd c:\Users\15864\Desktop\yunxixiaochengxu
docker compose up -d
```

验证容器状态：
```powershell
docker ps --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"
```

应看到 3 个容器均为 Up：
- shopxo-nginx: 0.0.0.0:8080->80/tcp
- shopxo-php: 9000/tcp
- shopxo-mysql: 0.0.0.0:3306->3306/tcp

### 2.2 验证后端可访问

```powershell
curl -s -o NUL -w "%{http_code}" http://localhost:8080/
```

预期返回：200

### 2.3 验证数据库连接

```powershell
docker exec shopxo-mysql mysql -uroot -proot123456 shopxo -e "SELECT 1;"
```

## 八、常见问题

### Q: Docker 容器启动失败
A: 检查端口占用：8080（Nginx）、3306（MySQL）

### Q: 后台提示无权限
A: 清除浏览器缓存，重新登录

### Q: 小程序无法请求后端
A: 
1. 确认后端运行：curl http://192.168.1.217:8080/
2. 确认勾选「不校验合法域名」
3. 确认 IP 正确：ipconfig | findstr "IPv4"

### Q: 数据库连接失败
A: 
1. 确认 MySQL 容器运行：docker ps
2. 确认密码：root123456
3. 从宿主机连接用 localhost:3306，容器间用 mysql:3306
