## 2. 上线前必验（安全与稳定性）

### 2.1 安全
- [ ] CORS_ORIGINS 已配置且不为空
- [ ] JWT_SECRET 已配置且满足强度要求（≥32字符）
- [ ] 生产环境不使用默认密码
- [ ] 生产环境不开启 SMOKE_TEST_BYPASS_CAPTCHA
- [ ] 生产环境不开启 WECHAT_REFUND_MOCK
- [ ] HTTPS 证书配置正确
- [ ] 微信回调签名验证正常

### 2.2 数据一致性
- [ ] 订单状态机转换正确（无非法跳转）
- [ ] 积分抵扣只减少 availablePoints，不减少 totalPoints
- [ ] 积分归还只增加 availablePoints，不增加 totalPoints
- [ ] 奖励积分同时增加 availablePoints 和 totalPoints
- [ ] PointsRecord.points 始终为正数
- [ ] 退款累计金额不超过实付金额

### 2.3 错误处理
- [ ] 错误码与 ERROR_CODE 常量一致
- [ ] 40001 参数错误
- [ ] 40102 Token 已过期
- [ ] 40103 Token 无效
- [ ] 40301 无权限
- [ ] 40401 资源不存在

### 2.4 CI 与测试
- [ ] GitHub Actions CI 通过
- [ ] API 单元测试通过（pnpm --filter @baby-mall/api test）
- [ ] API 构建通过（pnpm build:api）
- [ ] 管理后台构建通过（pnpm build:admin）
