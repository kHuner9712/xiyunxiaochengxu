# 项目结构说明

## 顶层目录

```
yunxixiaochengxu/
├── shopxo-backend/          # ShopXO 后端（PHP + ThinkPHP 8）
├── shopxo-uniapp/           # ShopXO uni-app 手机端（Vue 2 + uni-app）
├── docs/                    # 项目文档
├── docker/                  # Docker 配置
│   ├── nginx/               # Nginx 配置
│   │   └── default.conf     # Nginx 站点配置
│   └── php/                 # PHP-FPM 配置
│       └── Dockerfile       # PHP 镜像构建文件
├── docker-compose.yml       # Docker Compose 编排文件
└── .trae/                   # Trae IDE 配置
```

## 后端目录结构（shopxo-backend）

```
shopxo-backend/
├── app/                     # 应用代码
│   ├── admin/               # 管理后台模块
│   │   ├── controller/      # 控制器（Admin.php, Config.php, Goods.php 等）
│   │   ├── form/            # 表单验证
│   │   ├── lang/            # 多语言（zh/en/cht/spa）
│   │   └── view/            # 视图模板
│   ├── api/                 # API 模块（uni-app 对接入口）
│   │   ├── controller/      # API 控制器（User.php, Goods.php, Order.php 等）
│   │   ├── form/            # API 表单验证
│   │   └── config/          # API 路由配置
│   ├── index/               # 前台模块（PC/H5）
│   │   ├── controller/      # 前台控制器
│   │   ├── form/            # 前台表单验证
│   │   └── view/            # 前台视图模板
│   ├── install/             # 安装向导模块
│   │   ├── controller/      # 安装控制器
│   │   └── config/          # 安装临时配置（安装后可删除）
│   ├── service/             # 业务服务层（70+ Service 类）
│   ├── plugins/             # 插件目录
│   └── module/              # 模块组件（DIY、布局、表单）
├── config/                  # 系统配置
│   ├── app.php              # 应用配置
│   ├── database.php         # 数据库配置（安装后生成，gitignore）
│   ├── shopxo.php           # ShopXO 业务配置
│   ├── shopxo.sql           # 初始 SQL 文件
│   └── ...                  # 其他配置（cache/log/session 等）
├── extend/                  # 扩展类库
│   ├── base/                # 基础类（Email/Excel/Images/Wechat 等）
│   ├── payment/             # 支付扩展（微信/支付宝/银联等 40+ 支付方式）
│   └── qrcode/              # 二维码生成
├── public/                  # Web 根目录（Nginx 指向此目录）
│   ├── index.php            # 前台入口
│   ├── api.php              # API 入口（uni-app 调用）
│   ├── admin*.php           # 管理后台入口（安装后随机重命名）
│   ├── install.php          # 安装向导入口
│   ├── static/              # 静态资源（CSS/JS/图片）
│   └── download/            # 下载目录
├── runtime/                 # 运行时目录（日志/缓存）
├── vendor/                  # Composer 依赖
├── composer.json            # Composer 配置
└── .env                     # 环境变量（可选）
```

## Docker 环境结构

```
docker/
├── nginx/
│   └── default.conf         # Nginx 配置（rewrite 规则 + PHP-FPM 代理）
└── php/
    └── Dockerfile           # PHP 8.1-FPM + 扩展（pdo_mysql/mysqli/gd/zip/bcmath/fileinfo/redis）
```

### 容器说明

| 容器名 | 镜像 | 端口 | 说明 |
|--------|------|------|------|
| shopxo-nginx | nginx:1.25-alpine | 8080:80 | Web 服务器 |
| shopxo-php | 自构建 PHP 8.1-FPM | 9000 | PHP-FPM 应用服务 |
| shopxo-mysql | mysql:8.0 | 3306:3306 | MySQL 数据库 |

### 数据卷

| 卷名 | 挂载点 | 说明 |
|------|--------|------|
| shopxo-mysql-data | /var/lib/mysql | MySQL 数据持久化 |
