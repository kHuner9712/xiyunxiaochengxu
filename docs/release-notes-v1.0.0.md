## 四、已知限制

| 限制 | 说明 |
|------|------|
| 微信支付未配置 | 用户只能生成待支付订单，无法实际支付 |
| ICP 备案未完成 | 域名无法使用，小程序无法正式提审 |
| 使用测试 AppID | wxda7779770f53e901，正式提审需切换 |
| 活动报名免费 | 一期活动均为免费，无支付能力 |
| 敏感词精确匹配 | 不支持模糊/变体/拼音，后续可引入 NLP |
| 不回溯已有数据 | 内容合规扫描仅拦截新保存，不扫描已发布内容 |

## 七、回滚策略

| 场景 | 回滚方式 |
|------|----------|
| 体验版被拒 | 回到 release 分支修复，创建新 RC |
| 正式版发现 BLOCKER | 从 tag checkout 修复，打 v1.0.1 |
| 数据库迁移失败 | 执行回滚 SQL（需提前准备） |
| 服务器异常 | 回滚到上一个 commit，恢复 .env 配置 |
