## 二、验收步骤

### 2.1 登录后台

1. 浏览器打开 http://localhost:8080/adminwlmqhs.php
2. 输入用户名 `admin`，密码 `admin123`
3. 点击登录

**预期结果**：登录成功，跳转到后台首页

### 2.2 进入报名管理

1. 左侧菜单点击「运营」→「报名管理」
2. 可看到报名记录列表

**预期结果**：显示报名记录列表，包含姓名、手机、阶段、签到状态等列

### 2.3 执行签到

1. 找到一条签到状态为「未签到」的记录
2. 点击该记录的「签到核销」按钮
3. 确认签到

**预期结果**：
- 签到状态从「未签到」变为「已签到」
- 签到时间更新为当前时间

**数据库验证**（通过 MySQL 命令行）：
```sql
-- 查看签到前后变化
SELECT id, name, checkin_status, checkin_time, FROM_UNIXTIME(checkin_time) as checkin_time_text
FROM sxo_activity_signup WHERE id=3;
```

**已验证证据**（2026-04-17 真实测试）：
- 请求地址：`POST /adminwlmqhs.php?s=admin/activitysignup/checkin.html`
- 签到前：checkin_status=0, checkin_time=0
- 签到结果：msg="签到成功", code=0
- 签到后：checkin_status=1, checkin_time=1776374638 (2026-04-17 05:23:58)

### 2.4 导出报名数据

1. 在报名管理页面，点击「导出数据」按钮
2. 浏览器自动下载 CSV 文件

**预期结果**：
- 下载文件名格式：activity_signup_YYYYMMDDHHmmss.csv
- CSV 包含 11 列：报名ID、活动标题、姓名、手机、阶段、预产期、宝宝月龄、报名状态、签到状态、报名时间、签到时间
- CSV 使用 UTF-8 BOM 编码，Excel 可直接打开

**已验证证据**（2026-04-17 真实测试）：
- 请求地址：`GET /adminwlmqhs.php?s=admin/activitysignup/export.html`
- 导出数据条数：7 条
- 表头：报名ID,活动标题,姓名,手机,阶段,预产期,宝宝月龄,报名状态,签到状态,报名时间,签到时间
- 首行数据：ID=7, 阶段=备孕, 签到状态=已签到
